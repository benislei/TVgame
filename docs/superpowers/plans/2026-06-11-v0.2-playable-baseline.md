# v0.2 可玩基线整理 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将当前串流项目整理为 v0.2 可玩基线版，新增 HEVC 1080P60，并把推荐入口设为 HEVC 1080P30。

**Architecture:** 发送端继续通过 `RTP_PROFILES` 驱动画质档位，朋友包脚本只调用 profile 名称，不复制底层管线参数。Android 接收端已有 H.264/HEVC 动态解码能力，本次无需修改解码链路。

**Tech Stack:** Node.js、GStreamer、Android Java MediaCodec、PowerShell/Batch、node:test。
